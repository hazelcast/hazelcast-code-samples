How to run
----------

1. Build the project. </br>
For Windows
```
set JAVA_HOME=<jdk8 home path>
mvn clean install
```
For MacOS and Linux
```
export JAVA_HOME=<jdk8 home path>
mvn clean install
```
2. Start server
For Windows
```
set JAVA_HOME=<jdk17 home path>
start-server.bat
```
For MacOS and Linux
```
export JAVA_HOME=<jdk17 home path>
./start-server.sh
```
3. Start client

For Windows
```
set JAVA_HOME=<jdk8 home path>
start-client.bat
```
For MacOS and Linux
```
export JAVA_HOME=<jdk8 home path>
./start-client.sh
```
